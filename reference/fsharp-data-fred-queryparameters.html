QueryParameters Module

Namespace: FSharp.Data.Fred
Assembly: FSharp.Data.Fred.dll

Contains types for specifying search query parameters.

Types

Type Description

AggMethod

A key that indicates the aggregation method used for frequency aggregation. This parameter has no affect if the frequency parameter is not set.

Frequency

The FRED frequency aggregation feature converts higher frequency data series into lower frequency data series (e.g. converts a monthly data series into an annual data series). In FRED, the highest frequency data is daily, and the lowest frequency data is annual.

OrderByTags

Order Tags by values of the specified attribute.

SearchOrder

Order results by values of the specified attribute.

SearchType

Represents the type of search to perform.

SortOrder

Sort results is ascending or descending observation date order.

Units

A key that indicates a data value transformation.
